# Chobo

Get started with game programming using Ruby/Gosu as quickly as possible using Chobo!

## Installation

Install the gem

	gem install chobo

Create a new game

	chobo new my_awesome_game

This will create the following file structure

	.  
	|____assets
	|____config.rb
	|____content
	| |____audio
	| |____gfx
	|____game_window.rb
	|____lib
	| |____behaviors
	| |____game.rb
	|____my_awesome_game.rb

Run the game!

	ruby my_awesome_game.rb
